# Changelog

All notable changes to this project will be documented in this file.

---

## [1.4.0]
### Added
- Outil de formatage des données DIM siège (MCO, PSY, SMR) pour intégration dans PMSIpilot

### Fixed
- Mise à jour du format PMSIpilot avec support Vidhosp / Anohosp VA16
- Génération automatique des fichiers d'autorisations par champ
- Correction et génération des fichcomp MCO
- Amélioration de l’outil RAA : export supplémentaire par praticien + téléchargement en ZIP

---

## [1.3.1]
### Fixed
- Optimisation de l’outil de détection des doublons :
  - Passage de l’analyse année en cours → M-1 et M-2
  - Amélioration significative des performances

---

## [1.3.0]
### Added
- Outil de répartition des RUMs codés par TIM via interrogation BDD Oracle

### Changed
- Migration de l’application vers un serveur (disponibilité 24/7)
- Adaptation complète du code pour compatibilité Python 3.7 (downgrade depuis 3.10)

---

## [1.2.0]
### Added
- Outil de croisement de fichiers Excel pour le BHRe
- Outil de détection des doublons, séjours contigus et chevauchements (BDD Oracle)
- Outil de comparaison de tableaux de valorisation

---

## [1.1.0]
### Added
- Interface Streamlit avec menu de sélection d’outils
- Outil de découpage de fichiers Excel par chirurgien avec export ZIP
- Outil de visualisation des suppléments RUM via BDD Oracle
- Outil de croisement des RAA manquants (unité 60R)

---

## [1.0.0]
### Added
- Initialisation du projet Streamlit
