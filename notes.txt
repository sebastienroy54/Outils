# Portail Outils – Streamlit

## Objectif
Ce projet est un **portail d’outils internes** développé en **Streamlit**.  
Il centralise plusieurs outils métier nécessitant ou non une connexion Oracle.

Le portail permet :
- l’accès à différents outils depuis une interface unique
- la gestion d’une connexion Oracle partagée par session utilisateur
- l’exécution de requêtes SQL via SQL*Plus
- la génération et le téléchargement de fichiers (CSV / Excel)

---

## Environnement technique

- **OS hôte** : Debian 12
- **Docker** : 28.x
- **Python** : 3.7
- **Framework UI** : Streamlit
- **Accès Oracle** : SQL*Plus (Oracle Instant Client 10.2)
- **Connexion Oracle** : EZCONNECT
- **Reverse proxy** : aucun (accès direct par port)

---

## Déploiement

Le portail est exécuté dans un conteneur Docker avec :
- exposition directe d’un port TCP
- montage du code en volume pour permettre les modifications sans rebuild

L’application Streamlit écoute sur :
0.0.0.0:8501
